# fractales-portfolio
Este repositorio contiene dos demos interactivas de fractales:

mandelbox.html: Mandelbox 3D, demo WebGL2 con ray marching, selector de escenas y navegación suave.

mandelbrot.html: Fractal de Mandelbrot básico, renderizado en Canvas y controles de zoom.

📂 Estructura del proyecto

/ (raíz)
├── index.html          ← menú de demos
├── mandelbox.html      ← demo Mandelbox 3D 
├── mandelbrot.html     ← demo Fractal de Mandelbrot
├── assets/             ← carpeta de recursos
│   ├── js/             ← scripts JavaScript 
│   └── css/            ← hojas de estilo CSS
└── README.md           ← este archivo
